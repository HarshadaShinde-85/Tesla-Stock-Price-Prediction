# Tesla Stock Price Prediction using Machine Learning 🚗📈

This project predicts Tesla (TSLA) stock closing prices using Machine Learning models based on historical stock market data. It demonstrates a complete ML pipeline including data preprocessing, feature selection, model training, evaluation, and prediction visualization.
The objective of this project is to analyze Tesla stock trends and build predictive regression models to estimate future closing prices.

📊 Project Workflow
The project follows a structured ML lifecycle:
→ Data Collection
→ Data Cleaning
→ Data Preprocessing
→ Feature Selection
→ Model Training
→ Model Comparison
→ Prediction & Visualization

Dataset: Tesla historical stock price dataset (.xls)

Features used:

Date
Open price
High price
Low price
Volume
Close price (Target variable)

Target:  Close Price

⚙️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn
Jupyter Notebook

🧠 Machine Learning Models Used:
1️⃣ Linear Regression
Used as a baseline predictive model to estimate stock closing price.

2️⃣ Random Forest Regressor
Used for improved prediction accuracy by combining multiple decision trees.
