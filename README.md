# Tesla Stock Price Prediction using Machine Learning 🚗📈

This project predicts Tesla (TSLA) stock closing prices using Machine Learning models based on historical stock market data. It demonstrates a complete ML pipeline including data preprocessing, feature selection, model training, evaluation, and prediction visualization.
The objective of this project is to analyze Tesla stock trends and build predictive regression models to estimate future closing prices.

📊 Project Workflow
The project follows a structured ML lifecycle:
→ Data Collection
→ Data Cleaning
→ Data Preprocessing
→ Feature Selection
→ Model Training
→ Model Comparison
→ Prediction & Visualization
